# Contributing

Thanks for your interest in contributing to NoonFlow.

## Before You Start

- For reproducible bugs, please open an [Issue](https://github.com/heyallencao/NoonFlow/issues)
- For product ideas, workflow discussion, or prioritization, please use [Discussions](https://github.com/heyallencao/NoonFlow/discussions)
- For larger changes, open an issue or discussion first so the direction is clear before you spend time on implementation

## Pull Requests

- Keep pull requests focused and reasonably small
- Explain what changed and why
- Include screenshots if the change affects UI or onboarding
- Mention any manual testing you performed
- Avoid bundling unrelated refactors into the same pull request

## Scope

NoonFlow is still in an early public stage. Some product surfaces, naming, and structure may continue to evolve quickly.

That means:

- prioritize clear user value over broad refactors
- avoid over-claiming unfinished features in docs or product copy
- prefer changes that improve workflow quality, setup clarity, and day-to-day usability
- if public source snapshots are not yet present for a given area, use issues and discussions first before planning larger code contributions
