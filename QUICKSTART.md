## What You Need

- macOS
- Claude Code or Codex CLI installed
- Your runtime already authenticated

## Install

1. Download the latest build from [GitHub Releases](https://github.com/heyallencao/CairnStack/releases)
2. Move `CairnStack.app` into `/Applications`
3. Launch the app

If macOS blocks the app on first launch:

1. Right-click the app and choose **Open**
2. Go to **System Settings > Privacy & Security** and allow the app to run
3. If needed, run:

```bash
xattr -cr /Applications/CairnStack.app
```

## First Setup

1. Open Settings
2. Confirm your Claude Code or Codex runtime is available
3. Configure provider, model, or custom API endpoint if needed
4. Open a workspace and start a session
