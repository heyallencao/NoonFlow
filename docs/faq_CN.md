## CairnStack 是什么？

CairnStack 是一个面向 Claude Code 和 Codex 重度用户的可视化 AI coding 工作台。

## 它会替代 Claude Code 或 Codex 吗？

不会。CairnStack 是构建在这些运行时之上的工作台，而不是替代它们。

## 它是不是只是一个 wrapper？

它当然建立在底层 CLI 之上，但重点不是简单发命令，而是把 sessions、memory、skills、providers、costs 和项目可见性做得更强。

## 我还需要安装 Claude Code 或 Codex CLI 吗？

需要。当前公开版本依赖底层运行时已经安装并完成认证。

## 我可以接自定义 provider 或兼容中转的 API 吗？

可以。CairnStack 支持更灵活的 provider 和 model 配置，也适合接自定义 endpoint 和兼容中转的 API。

## 现在是不是只支持 macOS？

当前公开版本是的。后面会在把 macOS 版本打稳之后继续扩展更多平台。

## 什么样的反馈最有价值？

越具体越有价值：

- 你当时想做什么
- 哪些地方好用
- 哪些地方让你困惑
- 哪些地方坏了
- 你最希望下一步补什么
