# 快速开始

<p align="center">
  <a href="./QUICKSTART.md">English</a> | 简体中文
</p>

## 你需要准备

- macOS
- 已安装 Claude Code 或 Codex CLI
- 你的运行时已经完成认证

## 安装

1. 从 [GitHub Releases](https://github.com/heyallencao/NoonFlow/releases) 下载最新构建
2. 将 `NoonFlow.app` 移动到 `/Applications`
3. 启动应用

如果 macOS 在首次启动时阻止应用运行：

1. 右键点击应用并选择 **打开**
2. 前往 **系统设置 > 隐私与安全性**，允许应用运行
3. 如有需要，执行：

```bash
xattr -cr /Applications/NoonFlow.app
```

## 首次设置

1. 打开 Settings
2. 确认 Claude Code 或 Codex 运行时可用
3. 如有需要，配置 provider、model 或自定义 API endpoint
4. 打开一个 workspace 并开始会话
